# Gramo Statistikk-prototyper

Interaktive prototyper for statistikkvisning på Gramo sine flater.

## Oversikt

**[Se alle statistikk-prototyper →](index.html)**

Dette er prototyper som utforsker ulike innfallsvinkler til hvordan statistikk kan presenteres på Gramo.no.

## Publisert URL

Disse prototypene er tilgjengelig på:
**https://aregh.github.io/gramo-redesign-ny/**

## Om prototypene

Dette er **ideer til diskusjon**, ikke ferdige løsninger. Prototypene viser ulike konsepter for:
- Storytelling og narrativ presentasjon av data
- Sammenligning og VS-formater
- Kanal- og artistprofiler
- Interaktive filtre og søk
- ISRC-registrering og metadata
- Strømmekoplinger
- Geografiske visualiseringer
- og mye mer...

## Lokal visning

Åpne `index.html` i en nettleser for å se prototypene lokalt.
