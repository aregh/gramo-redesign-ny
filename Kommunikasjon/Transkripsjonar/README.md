# Transkripsjonar

> Råtranskripsjonar fra møter, intervjuer og workshops

---

## Formål

Denne mappen inneholder råtranskripsjonar som grunnlag for:
- Møtereferater
- Analyserapporter
- Innsiktsarbeid
- Dokumentasjon av beslutningsprosesser

---

## Filformat og navnekonvensjon

**Format:** `.md` (Markdown)  
**Navnekonvensjon:** `YYYY-MM-DD_Beskrivelse.md`

**Eksempler:**
- `2025-12-04_Intervju-Toril.md`
- `2025-11-18_Oppstartsworkshop.md`
- `2025-12-04_Møte-Kim-avklaring.md`

---

## Bruk av mal

Bruk `_MAL-Transkripsjon.md` som utgangspunkt for nye transkripsjonsfiler.

**Steg:**
1. Kopier `_MAL-Transkripsjon.md`
2. Gi filen riktig navn etter konvensjonen over
3. Fyll inn metadata (dato, deltakere, varighet)
4. Lim inn eller skriv transkripsjon
5. Strukturer med tidsstempler og overskrifter

---

## Retningslinjer

### Råtranskripsjon
- Behold original tekst så langt som mulig
- Marker uklare passasjer med `[uklart]`
- Marker pauser med `[pause]`
- Marker latter/reaksjoner med `[ler]`, `[sukk]` etc.

### Personvern
- Fjern sensitive personopplysninger
- Anonymiser hvis nødvendig
- Vurder om hele transkripsjon skal deles i repo

### Struktur
- Bruk tidsstempler for navigering
- Del inn i tematiske seksjoner
- Bruk **[Navn]:** for å markere hvem som snakker
